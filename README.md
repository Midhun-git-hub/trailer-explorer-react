# Trailer Explorer
A modern movie trailer discovery platform built with React and TMDB API.

# Tech Stack

- React
- React Router
- Tailwind CSS
- TMDB API
- YouTube Embed
- Vite
- JavaScript (ES6+)

# Explore Upcoming Movie Releases

- Watch Official YouTube Trailers
- Dedicated Movie Detail Page
- Pagination Support
- Skeleton Loading UI
- Modern Dark Theme
- Responsive 2-Column Grid Layout

# How It Works

- Fetches movie data from TMDB API
- Retrieves video data using /movie/{id}/videos
- Filters official trailers
- Embeds YouTube trailer directly inside the app

# API Notice

- This product uses the TMDB API but is not endorsed or certified by TMDB.
